COMPILE INSTRUCTIONS:
This program is written using Processing. 
To run the program please download the piPhiE.zip file and run in the Processing IDE. 

The necessary files include:
connection.pde
connnectionGraph.pde
piPhiE.pde

PROGRAM INSTRUCTIONS:
To select a view of pi, phi, or e: select the corresponding circular layout 
To go back to the view of all three options:  select the "BACK" option 
To load 100 more data points to the visualization: select the "Load Points" option
To change the edge colors: select the color palette options "1", "2", or "3"
To save an image: select the "SAVE" option
